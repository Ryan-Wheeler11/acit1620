What is Git?

Git is a version control system that lets you manage and keep track of your source code history


What is GitHub?

Github is proprietary developer platform that allows developers to create, store, manage, and share their code. GitHub is a cloud-based platform with its main prupose being facilitating version control and allowing collaboration for software development. It is built around Git, a popular version control system that helps developers track changes to code, coordinate work on projects, and manage code repositories.


How do Git and Github work together?

GitHub uses Git to keep track of every change made to a codebase.

Github's purpose is to host the Git system, aswell as add a few useful features.


Github allows us to...

  • Allows developers from all over the world to collobarate on different code or projects

  • Allows trunk based development

  • Developers can create branches, make changes, and merge updates without affecting the main project until the changes are tested and approved.


What is a Repository?

A repository is where project files and the history of their changes are stored.


How do you get started on GitHub?

  • Sign up for an account on GitHub
  • Create a repository for your project
  • Use Git to push code to GitHub
  • Explore colloboration tools, track changes or mistakes and enjoy!
